lubuntu配置 VNC

环境：mac , lubuntu

1.安装vnc服务

  • cyberdog开机,进入其命令行
1

ssh 192.168.2.180 -l mi
  • 安装tightvncserver
1

sudo apt install tightvncserver
  • 配置vncpasswd登录密码：
1
2
3

vncpasswd

Would you like to enter a view-only password (y/n)? n

启动 tightvncserver，生成默认配置文件

1

tightvncserver
  • 关闭：
1

tightvncserver -kill :1
  • 修改生成的配置文件
1

vim ~/.vnc/xstartup

替换以下内容

1
2
3
4
5
6

#!/bin/sh
xsetroot -solid grey
export XKL_XMODMAP_DISABLE=1
#autocutsel -fork
openbox &
/usr/bin/lxsession -s Lubuntu &
  • 开启vnc
1

tightvncserver :1
  • 查看启动日志：
1

cat ~/.vnc/lubuntu\:1.log
  • 默认端口：5901, 需要修改端口 参考文章最后👇

2.配置vnc连接

  • 下载 vnc viewer

https://www.guyuehome.com/35878

  • 打开vnc viewer 在最上的搜素框输入 自己的 ip:port 即可 连接成功。
  1. 配置自动启动 tightvncserver
  • 配置autostart 文件，自启动无法生效，原因未知，这里使用界面配置自启动
1
2
3

sudo vim /etc/xdg/lxsession/Lubuntu/autostart

@/usr/bin/tightvncserver :1

  • 界面配置自启动

    Preferences > Default applications > Manual autostarted Add

    1

    /usr/bin/tightvncserver :1

修改默认端口 （如果需要自定义端口，一般默认端口即可）

1

sudo vi /usr/bin/vncserver

修改218行，然后重启服务。

1
2
3
4

#关闭
tightvncserver -kill :1
#启动
tightvncserver :1
